# Решение задачи коммивояжера методом ветвей и границ

Десктопное приложение для точного решения задачи коммивояжера (TSP) с интерактивной визуализацией графа и найденного оптимального маршрута. Разработано в рамках дисциплины «Основы алгоритмизации и программирования» (ПНИПУ).

---

## Описание

Задача коммивояжера является классической NP-трудной задачей комбинаторной оптимизации. Данный проект реализует **точный алгоритм метода ветвей и границ**, который позволяет находить гарантированно оптимальный гамильтонов цикл минимального веса без полного перебора всех вариантов.

Программа предоставляет графический интерфейс для ввода весов рёбер через матрицу смежности, автоматическую отрисовку графа на сцене и наглядную подсветку найденного маршрута.

---

## Возможности

- **Точное решение:** Алгоритм Branch & Bound с эвристической нижней оценкой и отсечением неперспективных ветвей.
- **Интерактивный ввод:** Изменение матрицы смежности в таблице `QTableWidget` с мгновенной синхронизацией данных.
- **Умная визуализация:** 
  - Вершины располагаются равномерно по окружности.
  - Метки весов автоматически смещаются перпендикулярно рёбрам, предотвращая наложение текста.
  - Корректное управление слоями (`ZValue`): рёбра -> маршрут -> вершины -> текст.
- **Динамическое масштабирование:** Сцена автоматически подстраивается под размер окна.
- **Модульная архитектура:** Чёткое разделение логики, данных и отображения (MVC-подход).

## Архитектура

Проект построен по принципу разделения ответственности:

1. **Модель (`TSPGraph`)** — хранит матрицу смежности, управляет количеством вершин и весами рёбер.
2. **Алгоритм (`BranchAndBound`)** — реализует рекурсивный поиск пути с отсечением ветвей и расчётом нижних оценок.
3. **Визуализатор (`TSPVisualizer`)** — отвечает за отрисовку графа, расчёт координат вершин и управление слоями элементов на `QGraphicsScene`.
4. **Контроллер (`MainWindow`)** — связывает пользовательский интерфейс с логикой, обрабатывает сигналы кнопок и таблиц.
